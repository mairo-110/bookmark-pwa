# Local Bookmark PWA

URLを貼り付けるだけで端末内に保存する、iPhone向けのローカル専用ブックマークPWAです。

## 特徴

- データはブラウザの `localStorage` にのみ保存します。
- ブックマークはドメインごとに自動でグループ化されます。
- タイトルも保存して、共有ショートカットから受け取った内容を表示できます。
- タイトルは後から編集できます。
- お気に入りは星アイコンのタップで切り替えできます。
- JSONのエクスポートとインポートに対応しています。
- インストール可能な `manifest` と `service worker` を含みます。
- オフラインでも、保存済みのブックマークを閲覧できます。

## 使い方

1. `index.html` を HTTPS または `localhost` で配信します。
2. URL入力欄にブックマークしたいURLを貼り付けます。
3. 必要ならタイトルも入力して、`保存する` を押します。
4. iPhoneでは共有メニューからホーム画面に追加できます。

## データの保存

- `エクスポート` で全ブックマークを JSON で書き出せます。
- `インポート` で JSON を読み戻せます。
- お気に入り状態も一緒に保存されます。

## ショートカット連携

Safari の共有から iPhone のショートカットでこの PWA を開くときは、URL に `?url=...&title=...&save=1` を付けるか、`payload` に JSON を入れると、受け取ったタイトル付きで自動保存できます。

空白を含むタイトルは `payload` 方式が安全です。

`share` に `タイトル|URL` か `タイトル - URL` を入れる形式も使えます。

例: `index.html?payload=%7B%22url%22%3A%22https%3A%2F%2Fexample.com%2Farticle%22%2C%22title%22%3A%22記事 の タイトル%22%7D&save=1`

## 補足

- この環境では Node.js が利用できなかったため、ビルド工程は入れていません。
- 必要なら次の段階で Vite 版や IndexedDB 版に切り替えられます。
